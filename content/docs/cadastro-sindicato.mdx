## Introdução

Os acessos criados para acesso `entidades sindicais` serão enviados para nossa equipe de analise para verificação dos dados
para melhor atende-los, logo abaixo iremos abordar o passo a passo de como se registrar na plataforma e como ter acesso.

<Callout variant="info">
  **Informação:** Os acessos criados pela plataforma são totalmente segura e são
  gerados a partir da confirmação do e-mail, será enviado para sua caixa de
  entrada e pode demorar de 30 segundos á 5 minutos, caso não encontre em sua
  caixa de e-mail olhe em seu spam ou lixo eletrônico.
</Callout>

## Acesso

Para fazer o acesso basta acessar o portal [trabalhoavulso.com.br](http://trabalhoavulso.com.br/), acessar a área do cliente: igual a imagem abaixo:

1. Ao carregar a tela login, você seguirá os passos a seguir:

   - clicar em solicitar cadastro <sub>(igual imagem abaixo):</sub>

   - Ao selecionar `Entidade Sindical`, irá abrir outro modal de seleção do tipo de registro
     <sub>(igual imagem abaixo):</sub>
     <figure>
       <Image
         width={230}
         height={80}
         objectFit="cover"
         src="https://www.syng.com.br/Trabalhoavulso/Content/img/logo_syng_branco.png"
       />
       <figcaption>Modal tipo de registros</figcaption>
     </figure>
   - Existe 3 tipos de entidades sindicais:
     1. **Sindicato** -> Os Sindicatos contém uma base em uma determinada cidade e aborda uma região de
        suas proximidades, caso seja o seu caso selecione esta opção.
     2. **Federação** -> As Federações aborda um estado em si, caso seja o seu caso selecione esta opção.
     3. **Confederação** -> A Confederação aborda nacionalmente, caso seja o seu caso selecione esta opção.

   <Callout variant="warning">
     **Cadastros existentes:** Caso já exista uma entidade do tipo
     (Confederação), não será possível criar uma nova! Se algum estado já tiver
     cadastrado uma federação, não será possível criar uma nova para o mesmo
     estado! O mesmo aplica aos sindicatos quando uma cidade já estiver
     cadastrado um sindicato, não será possível criar um novo sindicato.
   </Callout>
